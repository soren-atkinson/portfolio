Rules

This page will be used to set the rules of the randomizer. You should be able to modify thing such as the stage of evolution, the type of the pokemon, and the generation. This will allow more customization of the randomizer and give the user more freedom in their experience. This page with have two button to access it with one being ont he header and the other being part the of page itself. This page with have multible setting to modify and the various pokemon will have tag on them that the setting will use when selection the pokemon.
